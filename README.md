# Max Group Fitness Capital

Base inicial para publicar el proyecto con GitHub Pages desde la rama main.

## Publicacion

1. Crear el primer commit:

```powershell
git add .
git commit -m "Initial GitHub Pages site"
git push -u origin main
```

2. En GitHub abrir:

- Settings
- Pages
- Build and deployment
- Source: Deploy from a branch
- Branch: main
- Folder: / (root)

3. Esperar el deploy y abrir la URL publicada por GitHub Pages.
